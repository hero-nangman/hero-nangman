## 소개

아이디어를 빠르게 제품으로 만들고 운영까지 이어가는 개발자입니다.  
MVP를 신속하게 출시한 뒤, 사용자 반응과 데이터를 바탕으로 서비스를 지속적으로 개선합니다.

- `Next.js + TypeScript`로 빠르고 완성도 높은 프론트엔드를 만듭니다
- `FastAPI + PostgreSQL`로 명확하고 확장 가능한 백엔드를 설계합니다
- 작은 단위로 빠르게 출시하고 실제 사용 데이터를 통해 검증합니다
- 사용자 피드백을 반영해 기능과 흐름을 지속적으로 개선합니다

## 작업 방식

> 빠르게 출시하고, 실제 사용자 반응으로 개선합니다.

- 문제를 작게 나누고 우선순위 높은 기능부터 구현합니다
- 초기에는 속도를 중시하되, 이후 확장을 고려한 구조를 유지합니다
- 기술 선택보다 사용자 문제 해결과 운영 효율을 우선합니다
